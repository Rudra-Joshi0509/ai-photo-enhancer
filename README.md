# ✨ AI Photo Enhancer

AI Photo Enhancer is a web-based application that enhances images using advanced image processing techniques.

## 🚀 Features
- Upload image
- Multiple enhancement modes:
  - Soft
  - Normal
  - Strong
  - Night Enhance
  - High Detail
  - Warm Tone
- Real-time processing
- Download enhanced image

## 🧠 Technologies Used
- Python
- OpenCV
- NumPy
- Streamlit
- Pillow
